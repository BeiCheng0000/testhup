## 修复 NameError: name 'models' is not defined
- **问题**: `apps/users/views.py` 的 `UserListView.get_queryset` 使用了 `models.Q()` 但未导入 `from django.db import models`
- **触发**: `GET /api/users/users/?search=...` 时报500错误
- **修复**: 在文件头部添加 `from django.db import models`

## 修复前端搜索用户报 TypeError: Cannot read properties of null (reading 'id')
- **问题**: `ProjectDetail.vue` 的 `searchUsers` 中 `userOptions.value = response.data`，但 DRF `ListAPIView` 默认分页返回 `{count, next, previous, results}` 对象而非数组
- **触发**: `v-for="user in userOptions"` 遍历分页对象时，`next`/`previous` 为 null，访问 `null.id` 报错
- **修复**: `userOptions.value = response.data?.results || response.data || []`
- 前端已重新构建，Nginx 已 reload
